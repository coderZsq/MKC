- [ ] **AC-1** 文章包含项目背景、架构设计、核心链路、关键难点、效果评估、部署与复盘
- [ ] **AC-2** 引用 README、架构图、评估报告、Grafana 看板截图或等价素材

## 技术要点

- 推荐标题聚焦“多媒体知识库助手”与“从 0 到上线”的工程过程。
- 每个亮点都关联到真实模块或文档，避免写成泛泛的大模型应用介绍。
- 配图使用本项目真实架构图、评估报告、监控看板或脱敏截图。
- 文章末尾给出技术栈、仓库结构、未来优化方向。
